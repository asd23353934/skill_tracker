# 🎮 技能追蹤器 v1.0.1

楓之谷 Artale 技能冷卻時間追蹤工具

## 📦 下載與安裝

### 下載
從 [Releases](https://github.com/asd23353934/skill_tracker/releases) 下載最新版本

### 安裝
1. 解壓縮 `技能追蹤器_v1.0.1.zip`
2. 雙擊 `技能追蹤器.exe` 即可運行
3. **不需要** 安裝 Python 或其他依賴

## 📁 文件結構

```
技能追蹤器/
├── 技能追蹤器.exe        ← 主程式
├── config.json           ← 技能配置（可編輯）
├── images/               ← 技能圖示
│   ├── bishop_holy_symbol.png
│   ├── snow_crystal.png
│   └── ...
├── profiles/             ← 配置檔案
│   └── 預設配置.json
└── _internal/            ← 程式依賴（自動生成）
```

## 🚀 快速開始

### 1. 首次運行
雙擊 `技能追蹤器.exe`

### 2. 設定快捷鍵
點擊 `[未設定]` → 按下快捷鍵（如 F1）→ 自動保存

### 3. 修改冷卻時間
點擊秒數按鈕 → 輸入新秒數 → 自動保存

### 4. 使用技能
按下快捷鍵 → 彈出倒數視窗 → 倒數結束提示音

## 💡 進階功能

### 配置管理
- 點擊 `[💾 配置管理]` 管理多個配置
- 適合多職業或不同場景切換

### 網路同步
- 創建房間 → 分享代碼給隊友
- 技能冷卻即時同步

### 自訂技能
編輯 `config.json`，在 `skills` 陣列中新增：

```json
{
  "id": "my_skill",
  "name": "我的技能",
  "icon": "my_skill.png",
  "cooldown": 120,
  "hotkey": "",
  "category": "player",
  "subcategory": "自訂"
}
```

將圖示 `my_skill.png` 放入 `images/` 資料夾

## ⚙️ 技能選項說明

- **發（綠）**：使用時發送給房間
- **收（藍）**：接收其他人的技能
- **駐（黃）**：視窗常駐不關閉

## 🎨 視覺提示

### 快捷鍵按鈕
- 🟡 黃色 = 已設定
- ⚫ 灰色 = 未設定

### 秒數按鈕
- 🔵 藍色 = 已修改
- ⚫ 灰色 = 原始值

## ❓ 常見問題

### Q: 快捷鍵沒反應？
A: 檢查是否被其他程式佔用，或關閉輸入法

### Q: 如何備份配置？
A: 複製整個 `profiles/` 資料夾

### Q: 如何恢復原始秒數？
A: 點擊秒數按鈕，輸入 config.json 中的原始值

### Q: 可以新增技能嗎？
A: 可以！編輯 `config.json` 並添加圖示到 `images/`

### Q: 打開程式後沒反應？
A: 檢查是否被防毒軟體阻擋，或以管理員身份運行
